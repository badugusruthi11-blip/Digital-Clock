# Digital Clock with Alarm System

## Project Overview
The **Digital Clock with Alarm System** is a Java desktop application that displays the current system time in real time and allows users to set an alarm.
The application is built using Java Swing for the graphical user interface (GUI) and Java's Date and Time API for time management.
The project demonstrates object-oriented programming concepts, event handling, multithreading, and GUI development in Java.

## Features
- Displays the current time in digital format.
- Updates the clock every second.
- Allows users to set an alarm time.
- Notifies the user when the alarm time is reached.
- Simple and user-friendly graphical interface.
- Uses Java Swing components.

## Technologies Used
- Java
- Java Swing
- AWT
- Java Date & Time API
- Multithreading

## How to Run
1. Download or clone the project.
2. Open the project in your Java IDE.
3. Compile all Java files.
4. Run `Main.java`.
5. The Digital Clock window will open.
6. Set the alarm time and wait for the notification.

## Expected Output
- A digital clock displaying the current system time.
- Alarm notification when the specified time is reached.
## Conclusion
The Digital Clock with Alarm System is a beginner-friendly Java project that demonstrates how real-time applications are developed using Java Swing and object-oriented programming.
It provides practical experience in GUI design, event handling, and time-based application development.
